### Install

```console

```

> Usage (Example)

```json
{
    "root": true,
    "parserOptions": {
        "ecmaVersion": 2018
    },
    "env": {
        "es6": true,
        "node": true
    },
    "extends": [
        "eslint:recommended",
        "@lxghtless/eslint-config-prettier"
    ]
}
```
